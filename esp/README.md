# PIN-Belegung ESP

GPIO14 - SCL Display
GPIO12 - SDA Display
GPIO13 - LED im Display
GPIO5 - Relay Dev-Board

GPIO16 - LED1 Wandgehäuse
GPIO4 - LED2 Wandgehäuse


# Display

Lila - SCL
Blau - SDA
Grau - GND
Weiß - Vcc
Schwarz - Vcc

# ESP-Dev-Board

Rot - GPIO4 (-> LED2)
Grün - GPIO13 (-> LED im Display)
Orange - GPIO16 (-> LED1)
Lila - GPIO14 (-> SCL)
Blau - GPIO12 (-> SDA)
Grau1 - GND (-> Display)
Grau2 - GND (-> LED im Display)
Schwarz - Vcc/3.3V
Gelb - GND (-> LED1/LED2)

# Kabelbelegung (Cat.5 ESP<->Display)

Braun - VCC
Braun/W - VCC
Blau - GND
Blau/W - GND
Orange - Anode LED
Orange/W - Kathode LED
Grün - SCL
Grün/W - SDA
